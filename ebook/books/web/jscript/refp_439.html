Window.history Property

Name

Window.history Property---the History of the Window

Availability

Navigator 2.0, Internet Explorer 3.0

Synopsis

window.history

Description

The history property contains a read-only reference to the History object of window, which may be any top-level window or frame. See the History object for further details.
