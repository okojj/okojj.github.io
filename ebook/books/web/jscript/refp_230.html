Math.LN10 Constant

Name

Math.LN10 Constant---mathematical constant loge10

Availability

Navigator 2.0, Internet Explorer 3.0

Synopsis

Math.LN10

Description

Math.LN10 is loge10, the natural logarithm of 10, which has a value of approximately 2.3025850929940459011.
