Reset.value Property

Name

Reset.value Property---the label of a Reset button

Availability

Navigator 2.0, Internet Explorer 3.0

Synopsis

reset.value

Description

The value property is a read-only string that specified the text that appears within the Reset button. It is specified by the VALUE attribute of the <INPUT> tag that created the button. If no VALUE attribute is specified, the default value is "Reset".
