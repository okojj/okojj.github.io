Packages.java Property

Name

Packages.java Property---root of the core Java language packages

Availability

Navigator 3.0

Synopsis

Packages.java

Description

The java property of the Packages object is a JavaPackage that contains each of the packages that comprise the core Java class library. These are packages such as java.lang, java.io, and java.applet.

Usage

You can use the "global" java property anywhere you would use Packages.java.
