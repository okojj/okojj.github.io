Text Search

The Java applet below provides a search engine for the entire CD-ROM. Unlike the searches within an individual book, this search only finds words in head elements (<H1>, etc.) and titles.

This search engine requires a Java-capable browser, such as Netscape or Internet Explorer.

This search engine, JHLSearch, was written by John Leach.

Having problems with this search engine under Internet Explorer? Visit Microsoft's Troubleshooting Java Problems in Internet Explorer page.
